# 🦝 LedgerCoon: Shadow Protocol
**Premium Offline CFO & Financial Mascot**

LedgerCoon to unikalny, sarkastyczny menedżer finansów osobistych, który dba o Twój portfel i... Twój humor.

### 🚀 Kluczowe Funkcje:
* **Ghost Future:** Symulacja wpływu wydatków na przyszłą płynność.
* **Sensory Cienia:**
    * *Shake to Hide:* Rozmaż dane jednym potrząśnięciem.
    * *Tilt to Stash:* Obróć telefon, by wejść do Skarbca.
* **The Shredder:** Interaktywna niszczarka uciążliwych długów.
* **3 Tryby UI:** Boss (On), Silk (Ona), Electric (Dziecko).

### 🛠️ Bezpieczeństwo:
Aplikacja działa 100% offline jako **PWA**. Twoje dane nigdy nie opuszczają urządzenia.
